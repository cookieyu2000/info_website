# Info Website

Personal portfolio website built with Create React App.

## Live Site

https://cookieyu2000.github.io/info_website

## Local Development

```bash
npm install
npm start
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Deploy (GitHub Pages)

```bash
npm run deploy
```

## Project Notes

- Content files: `src/content/zh.js`, `src/content/en.js`
- Deploy config: `package.json` `homepage`, `predeploy`, `deploy`
